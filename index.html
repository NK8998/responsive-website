The AD102 'Ada Lovelace' GPU is expected to feature 144 SMs, a 71% increase over the existing GA102 GPU and house a massive 18,432 CUDA core count.


The GeForce RTX 4090's GPU SKU is rumored to feature 16,128 CUDA cores which is a 50% increase in core count over the RTX 3090 but 14% lower cores than the full-fat AD102 configuration which may arrive later as the RTX 4090 Ti. It's the same situation this generation where the 'Ti' variant of the 3090 got the full-fat GA102 GPU and the non-Ti variant, although launched earlier, came with a cut-down config.

In addition to the SM counts, the Ada Lovelace GPUs will also feature increased L2 cache sizes. Starting with the AD102 GPU, the flagship would be outfitted with up to 96 MB of L2 cache, an insane 16x increase over the 6 MB L2 cache featured on GA102. This cache still isn't as big as Infinity Cache featured on AMD's RDNA cards but it's still a huge leap for NVIDIA's own graphics cards.
